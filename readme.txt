【機能】
処理フォルダに入っている画像ファイル名を"yyyy-mm-dd_hhmm"形式にフォーマットする。

【仕様】
jpg/jpeg/png形式に対応。
処理対象のファイルと完了後の出力フォルダ内のフォルダをログとして出力する。
リネームの結果ファイル名が重複となる場合、末尾に連番を付与する。

・jpg/jpegについて
jpg/jpegはファイルプロパティからネーミング用の情報を取得する。
プロパティは撮影日→撮影日がない場合は更新日の順で取得する。

・pngについて
pngはファイル名に含まれる日時情報から正規表現で情報を取得してフォーマットする。(プロパティがないため)
正規表現のパターンに一致しないファイルは処理対象として取得しない。
新しいパターンを使用する場合はソース先頭付近にあるaddメソッドを使用して正規表現パターンを追加する。

【使い方】
1.inputsフォルダを作成し、対象の画像ファイルを入れる
2.jpgまたはpng用のスクリプト本体を起動する。
3.outputsに拡張子ごとの出力フォルダへリネーム後のファイルが移動される。
